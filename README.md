# Programa-Control_Horas
Solución en Python para el cálculo y clasificación de jornadas laborales usando matrices. Evaluación Final (Fase 5) - Fundamentos de Programación UNAD.

Evaluación Final - Fase 5: Fundamentos de Programación
Este repositorio contiene la solución al **Problema 5** del banco de problemas asignado para la evaluación final del curso Fundamentos de Programación de la UNAD
El script procesa una matriz bidimensional que registra las horas trabajadas por un equipo de cuatro recursos de lunes a viernes

La lógica del programa se encarga de:
1. Calcular la suma total de horas semanales por cada recurso
2. Evaluar el total de horas contra un umbral de 40 horas
3. Clasificar la jornada como "Sobretiempo" si supera el umbral, o "Horario Estándar" si se mantiene por debajo o igual[cite: 63, 64].

Requisitos Técnicos
* El código está escrito en **Python**
* Sigue el paradigma de programación estructurada utilizando funciones, ciclos y listas (matrices)
